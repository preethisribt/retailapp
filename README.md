# 🛒 Retail Product API (Spring Boot)

A backend project built using Spring Boot to simulate a simple retail system similar to Amazon.

This project is currently in development.

---

## 🚀 Current Features

- Product entity created
- REST APIs for:
    - Get all products
    - Get product by ID
- JPA + MySQL integration
- Global exception handling

---

## ⚙️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

---

## 🏗️ Work in Progress

Planned features:
- Create / Update / Delete product APIs
- DTO layer
- Validation
- Swagger documentation
- Pagination & filtering
- Authentication (Spring Security)

---

```bash
mvn spring-boot:run
